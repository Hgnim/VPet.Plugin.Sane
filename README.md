# 理智系统插件
> VPet.Plugin.Sane\
> 虚拟桌宠模拟器插件

为桌宠添加了理智值，理智值会根据桌宠的工作、休息、娱乐等行为进行变化，理智值过低会影响桌宠的健康。
- 理智值将显示在桌宠状态栏中
- 理智值会随着桌宠工作与学习逐渐降低
- 可以通过玩耍或睡觉来恢复理智值
- 可以通过食用增加健康值的食物来恢复理智值，但恢复的理智值是临时的，临时理智值会随着时间逐渐流失
- 什么都不做也能缓慢恢复理智值
- 理智值过低时会阶梯式降低健康值

> 桌宠也需要劳逸结合呢:D
